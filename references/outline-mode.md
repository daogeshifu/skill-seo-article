# Outline Mode

Use this mode when the user wants an outline, H2/H3 plan, or heading structure only.

## Output Shape

- Return JSON only
- Use `title` for 3 SEO-friendly title options
- Use `result` for a newline-separated heading plan

## Heading Rules

- Prefix every heading with `H2:` or `H3:`
- Never use markdown headings such as `##` or `###`
- Never output body copy
- Never output H4 or deeper

## Required Sections

- Include one or more topical H2 sections
- Include `H2: Conclusion`
- Include `H2: FAQs`
- Under FAQs, include 3 to 5 `H3:` question headings

## Adaptive Patterns

- Comparison keywords: explain each side, differences, decision guide
- Type or method keywords: create 3 to 5 H3 items under the relevant H2
- Cost keywords: cost drivers, alternatives, hidden costs
- Process keywords: preparation, steps, mistakes, safety
- Compliance keywords: types, frequency, standards, qualifications, checklist
